Selami Şahin

kitledim kalbimi al anahtarını
giderken boynuna takıver de git
öyle mahzun mahzun durma ne olur
yüzüme son defa gülüver de git

ayrılık saati çaldı çalacak
gözümden yaşları siliver de git

dudaklarım seni öpmeye hasret
beni son bir defa öpüver de git

seni dünyalarca seviyorum ben
aşkımıza yemin ediver de git
sensiz ölecekmiş gibiyim sanki
gel de canıma can katıver de git

sensiz gözlerimde herşey karanlık
gönlümde bir ışık yakıver de git
aydan yıldızlardan sana taç yaptım
giderken başına takıver de git
