Kara Bahçe

Demet Sağıroğlu

gün doğuyor yavaş yavaş
bilmem hangi bahçede var savaş
benim bahçem bir kara bahçe
Ah neden var bu bahçede savaş

gülmek herşeyi önlemez ki
neden durmadan gülüyorsun
hoşçakal gülüm sana elveda
sen gül, ben kara bahçedeyim.


git git hadi git sen git
ben burada kara bahçedeyiim!
belki gelir bir gün anlarsın,
niye gittin canım diye ağlarsın.

eğer bu olursa
yanıma gelmek istersen
ben kara bahçemdeyim
hüzünlerle savaşıyorum

hadi gel canım hadi gel gel
ben burada kara bahçedeyim
Ahhh... Kara bahçe ağlama yine gül
önlermiş gibi
