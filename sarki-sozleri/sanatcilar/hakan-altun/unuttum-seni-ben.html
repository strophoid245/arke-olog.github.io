Unuttum Seni Ben

Hakan Altun

sus konuşma artık seni dinlemem
verdigim sevgiyi yok ettin ellerinle
git başımdan daha çok üzme beni
kararımı verdim olmayacak
dönüşü olmaz aşkın yıkarken düşünseydin
pişmanlık fayda etmez sen istedin
unuttum seni ben günahlar boynuna
yazılmış yazıyı çevirdin tersine
Hani sen benimdin durmadın sozunde
istediğin yere git unuttum seni ben
aylardır bekledim haberin gelmedi
ansızın karşıma çıkmışsın nafile
sevgimi kaybettin mutluluk uzakta
istediğin yere git unuttum seni ben
bebegim günahlar tek senin boynuna
yazılmış yazıyı çevirdin sen tersine
hani sen benimdin durmadın sözünde
istediğin yere git unuttum seni ben
