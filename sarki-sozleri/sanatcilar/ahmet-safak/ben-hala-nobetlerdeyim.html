Ben Hala Nöbetlerdeyim

Ahmet Şafak

Ben bunu haketmedim
Ne varsa aşka ve cesarete dair
Sırlayıp o büyük yangınınla git

Bir şey duydum güzel yarim beni bırakıp gitmişsin
Delikanlı yüreğime hain kurşun sıkıp gitmişsin
Verdiğin sözü unutmuşsun mektupları yakıp gitmişsin
Ha benim kınalı kuzum ciğerimi söküp gitmişsin
Git ey yar en hain gülüşünle git
Git ey yar beni öldürüşünle git
Git ey yar ben hala
Nöbetlerdeyim ben hala hasretlerdeyim


Kınalı kuzum bu şarkıyı sana armağan ettim
Giderken yanına almayı sakın unutma
Belki soban sönmüş kitabın bitmiş
Belki dizlerinde battaniye
Yalnızlığın iç çekişini duyarsın
Paketteki son sigara ve
Titrek bir mum alevi hüznüyle
Geçmişe dalarsın ve kimseler görmeden sende kanarsın
Hayatımın baharına ecel gibi çöküp gitmişsin
Bulutları kirpiğime zamansız döküp gitmişsin
Gezdiğimiz o yollara hasreti ekip gitmişsin
Ayrılığın çarmıhına kalbimi çakıp gitmişsin...
