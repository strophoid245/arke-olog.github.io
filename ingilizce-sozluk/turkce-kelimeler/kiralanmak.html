kiralanmak

1. /ı, a/ (for a house, apartment, office, etc.) to be rented, Brit. be let (to); (for a car, boat, horse, machine, etc.) to be rented (out) (to), Brit. be hired (out) (to); (for something) to be leased (to). 2. /ı, dan/ (for a house, apartment, office, etc.) to be rented (from); (for a car, boat, horse, machine, etc.) to be rented, Brit. be hired (from); (for something) to be leased (from); (for a boat, bus, plane, etc.) to be chartered (from).
