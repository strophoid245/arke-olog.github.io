beher

each: Beher gülün fiyatı yedi bin lira. Each rose costs seven thousand liras. ––i each one, each: Eğer yüz tane alırsanız, beheri size on bin liraya mal olur. If you buy a hundred, each one will cost you ten thousand liras. ––ine for each one, apiece: Selim Bey beherine beş bin lira verecek. Selim Bey will pay five thousand liras apiece.
