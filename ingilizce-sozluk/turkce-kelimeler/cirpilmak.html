1. (for a tablecloth, rug, etc.) to be shaken out. 2. (for a piece of laundry, a cloth, etc.) to be agitated (in water) (as is done when rinsing). 3. (for hands) to be clapped. 4. (for wings) to be flapped. 5. to be trimmed, be clipped. 6. (for a food) to be beaten, whisked, or whipped. 7. to be stolen.
