kundak

1. cloths used for swaddling, swaddling clothes. 2. gunstock. 3. bundle of oily rags (used by an arsonist to start a fire). –– bezi cloth used for swaddling. ––tan çıkmak to cease to need to be swaddled. –– çocuğu/––taki çocuk a babe in arms. –– sokmak/koymak 1. /a/ (for an arsonist) to set fire to (a place) using a bundle of oily rags. 2. /arasına/ to set (two people) against (each other). 3. /a/ to sabotage, wreck (a project).
